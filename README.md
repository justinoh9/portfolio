## Tech

Vanilla HTML/CSS/JS — no framework, no build step. All site copy lives in [`content.js`](content.js); the pages render it client-side. Project thumbnails are hand-made SVGs in [`assets/thumbs/`](assets/thumbs/). Live GitHub stats are pulled from the GitHub API at page load.

## Editing

- Update text/projects/experience → edit `content.js`
- Add a project image → drop it in `assets/` and set the project's `image` field
- Resume → replace `assets/Justin-Oh-Resume.pdf`
